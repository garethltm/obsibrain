- DB Management process implemented via [[software]]
### Features
1. Data Storage & Integrity
2. Data Security & Access
3. Retrieval & Updates (usually Concurrent)
4. Backup & Recovery
5. Communication Interfaces (GUI, API, Web service, etc.)

### Pros
- Control Redundancy
- Data sharing
- Central Maintenance
- Faster Development
- Business Continuity
- Multiple Interfaces
- Data Partitioning

### Cons
- Cost of Hardware
- Cost of [[software]]
    - License vs Free
- Specialised Skill
- Single Point of Failure
    - Data Corruption
    - Hacking
